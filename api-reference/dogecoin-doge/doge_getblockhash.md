# getblockhash - Dogecoin

#### Parameters

`index` - integer

index of the block.

#### Request

```java
curl --location --request POST 'https://go.getblock.io/<ACCESS-TOKEN>/' \
--header 'Content-Type: application/json' \
--data-raw '{"jsonrpc": "2.0",
"method": "getblockhash",
"params": [3904788],
"id": "getblock.io"}'
```
