# getmininginfo - Litecoin

Returns a json object containing mining-related information.

### Parameters

* None

### Request

{% tabs %}
{% tab title="cURL" %}
{% code title="cURL" %}
```bash
curl --location --request POST 'https://go.getblock.io/<ACCESS-TOKEN>/' \
--header 'Content-Type: application/json' \
--data-raw '{
    "jsonrpc": "2.0",
    "method": "getmininginfo",
    "params": [],
    "id": "getblock.io"
}'
```

```javascript
import axios from 'axios';

    "method": "getmininginfo",
    "params": [],
    "id": "getblock.io"
});

    method: 'post',
    url: 'https://go.getblock.io/<ACCESS-TOKEN>/',
    headers: {
        'Content-Type': 'application/json'
    },
    data: data
};

axios(config)
    .then(response => console.log(JSON.stringify(response.data)))
    .catch(error => console.log(error));
```

{% tab title="Python (requests)" %}
{% code title="request.py" %}
```python
import requests
import json

url = "https://go.getblock.io/<ACCESS-TOKEN>/"

payload = json.dumps({
    "jsonrpc": "2.0",
    "method": "getmininginfo",
    "params": [],
    "id": "getblock.io"
})

headers = {
    'Content-Type': 'application/json'
}

response = requests.post(url, headers=headers, data=payload)
print(response.text)
```

```rust
        .header(header::CONTENT_TYPE, "application/json")
        .body(r#"{

            "method": "getmininginfo",
            "params": [],
            "id": "getblock.io"
        }"#)
        .send()

    println!("{}", response.text().await?);
    Ok(())
}
```

```json
    "jsonrpc": "2.0",

    "result": {
        "blocks": 2750000,
        "difficulty": 12345678.90,
        "networkhashps": 123456789012345,
        "pooledtx": 100,
        "chain": "main",
        "warnings": ""
    }
}
```
{% endcode %}

### Response Parameters

| Field         | Type   | Description                    |
| ------------- | ------ | ------------------------------ |
| blocks        | number | The current block.             |
| difficulty    | number | The current difficulty.        |
| networkhashps | number | The network hashes per second. |
| pooledtx      | number | The size of the mempool.       |

### Use Case

The `getmininginfo` method is essential for:

* Mining pool dashboards
* Network statistics
* Hashrate monitoring
* Mining profitability calculators

### Error Handling

| Status Code | Error Message | Cause                            |
| ----------- | ------------- | -------------------------------- |
| 403         | Forbidden     | Missing or invalid ACCESS-TOKEN. |
