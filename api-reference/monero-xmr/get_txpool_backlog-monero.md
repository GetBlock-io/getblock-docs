# get\_txpool\_backlog monero

This method returns the current backlog of transactions in the mempool, including projected confirmation times.

## Parameters

* None

## Response Parameters

| Field                            | Type         | Description                                   |
| -------------------------------- | ------------ | --------------------------------------------- |
| result.backlog                   | array        | Array of mempool transactions                 |
| result.backlog\[].blob\_size     | unsigned int | Serialized transaction size in bytes          |
| result.backlog\[].weight         | unsigned int | Transaction weight (used for fee calculation) |
| result.backlog\[].fee            | unsigned int | Transaction fee in atomic units               |
| result.backlog\[].time\_in\_pool | unsigned int | Seconds the transaction has been in the pool  |

## Use Cases

* Mempool congestion monitoring
* Fee oracle inputs
* Wallet UX showing confirmation-time estimates
