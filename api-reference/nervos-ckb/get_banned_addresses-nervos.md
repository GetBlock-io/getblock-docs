# get\_banned\_addresses - Nervos

Returns the IP addresses currently banned by the node — their network addresses, ban reason, ban timestamp, and ban expiration.

## Parameters

* None

## Response Parameters

| Field                 | Type                | Description                                             |
| --------------------- | ------------------- | ------------------------------------------------------- |
| result                | array of BannedAddr | Banned addresses                                        |
| result\[].address     | string              | Network address (IP/CIDR or peer ID)                    |
| result\[].ban\_reason | string              | Reason for the ban                                      |
| result\[].ban\_until  | Uint64              | Unix timestamp in milliseconds when the ban expires     |
| result\[].created\_at | Uint64              | Unix timestamp in milliseconds when the ban was created |

## Use Cases

* Auditing peer ban activity
* Debugging connection issues caused by accidental self-ban
* Network security monitoring

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| -32603      | Internal error    | Server-side error while processing the request       |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
