# get\_blockchain\_info - Nervos

Returns statistics about the chain — chain name, current median time, current epoch, current difficulty, whether the node is in Initial Block Download, and any active chain alerts.

## Parameters

* None

## Response Parameters

| Field                               | Type                    | Description                            |
| ----------------------------------- | ----------------------- | -------------------------------------- |
| result.chain                        | string                  | Chain identifier — `"ckb"` for mainnet |
| result.median\_time                 | Uint64                  | Past median timestamp (ms hex)         |
| result.epoch                        | EpochNumberWithFraction | Current epoch                          |
| result.difficulty                   | Uint256                 | Current chain difficulty               |
| result.is\_initial\_block\_download | boolean                 | `true` if node is still syncing        |
| result.alerts                       | array of Alert          | Active chain alerts                    |

## Use Cases

* Comprehensive single-call health checks
* Confirming network identity (mainnet vs testnet)
* Difficulty monitoring

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| -32603      | Internal error    | Server-side error while processing the request       |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
