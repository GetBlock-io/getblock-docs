# get\_fee\_rate\_statistics - Nervos

Returns fee rate statistics over recent blocks — mean, median, and p95 fee rates. Use to construct a fee oracle for transaction submissions.

## Parameters

| Parameter | Type   | Required | Description                                           |
| --------- | ------ | -------- | ----------------------------------------------------- |
| target    | Uint64 | No       | Number of recent blocks to consider; omit for default |

## Response Parameters

| Field         | Type   | Description                        |
| ------------- | ------ | ---------------------------------- |
| result.mean   | Uint64 | Mean fee rate (Shannon per byte)   |
| result.median | Uint64 | Median fee rate (Shannon per byte) |

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| -32603      | Internal error    | Server-side error while processing the request       |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
