# get\_indexer\_tip - Nervos

Returns the indexer's tip — the block hash and number that the built-in indexer has finished processing. May lag behind the chain tip during catch-up.

## Parameters

* None

## Response Parameters

| Field                | Type   | Description                        |
| -------------------- | ------ | ---------------------------------- |
| result.block\_hash   | H256   | Hash of the latest block indexed   |
| result.block\_number | Uint64 | Number of the latest block indexed |

## Use Cases

* Detecting indexer lag relative to chain tip
* Routing indexer-dependent reads only when caught up
* Indexer health probes

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| -32603      | Internal error    | Server-side error while processing the request       |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
