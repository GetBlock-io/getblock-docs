# eth\_chainid tempo

Returns the chain ID of the connected network. For Tempo mainnet the result is `0x1079` (4217 in decimal); for the Moderato testnet it is `0xa59f` (42431). Chain ID is required for EIP-155 signed transactions to prevent replay attacks across chains.

## Parameters

* None

## Response Parameters

| Field  | Type   | Description                                                                          |
| ------ | ------ | ------------------------------------------------------------------------------------ |
| result | string | Chain ID in hexadecimal (`0x1079` = 4217 mainnet, `0xa59f` = 42431 Moderato testnet) |

## Use Cases

* Constructing EIP-155 signed transactions
* Validating that a wallet is connected to Tempo
* Multi-chain dApps that branch logic by chain ID

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
