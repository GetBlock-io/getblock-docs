# eth\_blocknumber worldchain

This method retrieves the latest block number from the current chain head, offering real-time insight into the blockchain's progress. This method is essential for applications that need to stay updated with the latest state of the World Chain blockchain.

### Parameters

* None

### Request

{% tabs %}
{% tab title="curl" %}
```bash
curl --location --request POST 'https://go.getblock.io/<ACCESS-TOKEN>/' \
--header 'Content-Type: application/json' \
--data-raw '{
    "jsonrpc": "2.0",
    "method": "eth_blockNumber",
    "params": [],
    "id": "getblock.io"
}'
```
{% endtab %}

### Response

```json
{
    "jsonrpc": "2.0",
    "id": "getblock.io",
    "result": "0x1A5C3B2"
}
```

### Response Parameters

| Field  | Type   | Description                                            |
| ------ | ------ | ------------------------------------------------------ |
| result | string | Hexadecimal representation of the current block number |

### Use Case

The `eth_blockNumber` method on World Chain is typically used for:

* Monitoring blockchain progress
* Synchronization status checks
* Block height tracking for dApps
* Transaction confirmation monitoring

### Error Handling

| Status Code | Error Message | Cause                           |
| ----------- | ------------- | ------------------------------- |
| 403         | Forbidden     | Missing or invalid ACCESS-TOKEN |
