# eth\_gasprice - Worldchain

This method returns the current gas price on the World Chain network in wei. This method is essential for estimating transaction costs.

### Parameters

* None

### Request

{% tabs %}
{% tab title="curl" %}
```bash
curl --location --request POST 'https://go.getblock.io/<ACCESS-TOKEN>/' \
--header 'Content-Type: application/json' \
--data-raw '{
    "jsonrpc": "2.0",
    "method": "eth_gasPrice",
    "params": [],
    "id": "getblock.io"
}'
```
{% endtab %}

{% tab title="JavaScript (axios)" %}
```javascript
import axios from 'axios';

const data = JSON.stringify({
    "jsonrpc": "2.0",
    "method": "eth_gasPrice",
    "params": [],
    "id": "getblock.io"
});

const config = {
    method: 'post',
    url: 'https://go.getblock.io/<ACCESS-TOKEN>/',
    headers: {
        'Content-Type': 'application/json'
    },
    data: data
};

axios(config)
    .then(response => console.log(JSON.stringify(response.data)))
    .catch(error => console.log(error));
```

use reqwest::header;

### Response

```json
{
    "jsonrpc": "2.0",
    "id": "getblock.io",
    "result": "0x3B9ACA00"
}
```

### Response Parameters

| Field  | Type   | Description                                                |
| ------ | ------ | ---------------------------------------------------------- |
| result | string | Hexadecimal representation of the current gas price in wei |

### Use Case

The `eth_gasPrice` method on World Chain is typically used for:

* Transaction fee estimation
* Gas price monitoring
* Dynamic fee adjustment
* Cost optimization

### Error Handling

| Status Code | Error Message | Cause                           |
| ----------- | ------------- | ------------------------------- |
| 403         | Forbidden     | Missing or invalid ACCESS-TOKEN |
