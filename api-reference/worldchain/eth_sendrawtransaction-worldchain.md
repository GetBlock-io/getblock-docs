# eth\_sendrawtransaction - Worldchain

Submits a pre-signed transaction for broadcast to the World Chain network.

### Parameters

| Parameter             | Type   | Description                               |
| --------------------- | ------ | ----------------------------------------- |
| signedTransactionData | string | The signed transaction data (hex encoded) |

### Response

```json
{
    "jsonrpc": "2.0",
    "id": "getblock.io",
    "result": "0xe670ec64341771606e55d6b4ca35a1a6b75ee3d5145a99d05921026d1527331"
}
```

### Response Parameters

| Field  | Type   | Description                                   |
| ------ | ------ | --------------------------------------------- |
| result | string | Transaction hash of the submitted transaction |

### Use Case

The `eth_sendRawTransaction` method on World Chain is typically used for:

* Transaction broadcasting
* Payment submission
* Contract interaction
* Token transfers

### Error Handling

| Status Code | Error Message | Cause                           |
| ----------- | ------------- | ------------------------------- |
| 403         | Forbidden     | Missing or invalid ACCESS-TOKEN |
