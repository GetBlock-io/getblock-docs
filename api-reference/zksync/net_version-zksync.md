# net\_version zksync

Returns the current network ID as a decimal string. For zkSync Era mainnet: `"324"`; Sepolia testnet: `"300"`. Decimal string format, unlike `eth_chainId` which returns hex.

## Parameters

* None

## Response Parameters

| Field  | Type   | Description                                                   |
| ------ | ------ | ------------------------------------------------------------- |
| result | string | Network ID as a decimal string (`324` for zkSync Era mainnet) |

## Use Cases

* Detecting which network the node is on
* Legacy compatibility with tools using `net_version` over `eth_chainId`

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
