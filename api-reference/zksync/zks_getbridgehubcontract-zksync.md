# zks\_getBridgehubContract - zkSync

Returns the L1 bridgehub contract address. The bridgehub is the central coordinator for all canonical bridge operations across zkSync hyperchains.

## Parameters

* None

    "method": "zks_getBridgehubContract",
    "params": [],

use reqwest::Client;
use serde_json::json;

#[tokio::main]
async fn main() -> Result<(), Box<dyn std::error::Error>> {
    let client = Client::new();

    let payload = json!({
        "jsonrpc": "2.0",
        "method": "zks_getBridgehubContract",
        "params": [],
        "id": "getblock.io"
});

    let response = client
        .post("https://go.getblock.io/<ACCESS-TOKEN>/")
        .header("Content-Type", "application/json")
        .json(&payload)
        .send()
        .await?;

    let result: serde_json::Value = response.json().await?;
    println!("{:#?}", result);

    Ok(())
}

## Response Parameters

| Field  | Type   | Description                                  |

## Use Cases

* Bridge integrations querying the canonical bridgehub address

## Error Handling

| Status Code | Error Message     | Cause                               |
| ----------- | ----------------- | ----------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>` |
