# zks\_getl1batchblockrange zksync

Returns the range of L2 blocks contained within a given L1 batch. The range is returned as a two-element array `[firstBlock, lastBlock]` in hex.

## Parameters

| Parameter     | Type    | Required | Description                  |
| ------------- | ------- | -------- | ---------------------------- |
| l1BatchNumber | integer | Yes      | The L1 batch number to query |

## Response Parameters

| Field  | Type            | Description                                                                  |
| ------ | --------------- | ---------------------------------------------------------------------------- |
| result | array of string | Array `[firstBlock, lastBlock]` in hex — the L2 block range in this L1 batch |

## Use Cases

* Mapping L2 blocks to their containing L1 batch
* Batch-level explorers and analytics
* Replaying batch contents during proof verification

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
