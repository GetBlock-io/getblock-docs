# zks\_getl1gasprice zksync

Returns the current L1 gas price used in fee calculations. The L1 gas price contributes to L2 transaction costs since pubdata posting to L1 is part of every transaction's fee.

## Parameters

* None

## Response Parameters

| Field  | Type   | Description                               |
| ------ | ------ | ----------------------------------------- |
| result | string | Current L1 gas price in wei per gas (hex) |

## Use Cases

* L1→L2 fee estimation
* Building fee oracles aware of L1 conditions
* Analytics on L1 settlement cost components

## Error Handling

| Status Code | Error Message     | Cause                                                |
| ----------- | ----------------- | ---------------------------------------------------- |
| 403         | Forbidden         | Missing or invalid `<ACCESS-TOKEN>`                  |
| -32602      | Invalid params    | Request parameters are missing or malformed          |
| -32601      | Method not found  | Method does not exist or is not enabled on this node |
| 429         | Too Many Requests | Rate limit exceeded for your plan                    |
